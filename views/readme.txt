

This is a personal blog developed using JavaScript, EJS, and Node.

 It is a simple blogging platform with the ability to create, 
 edit, and delete posts. 
 
 The home page displays a brief description of the blog, 
 while the About and Contact pages contain standard information 
 about the author and how to contact them, respectively.
 
 The Compose page provides a form to create a new post, 
 which will be displayed on the home page along with previously created posts. 
 
 The blog also includes a feature to display individual blog posts when their title is clicked on.
